## 🚀 What is StackZen?

StackZen is an AI-powered engineering knowledge platform that helps developers instantly understand projects.

Instead of searching through documentation, API specifications, database schemas, meeting notes, and repositories, developers can simply ask questions in natural language and receive accurate, source-backed answers.

---

## ✨ Our Mission

Reduce the time developers spend searching for information so they can spend more time building software.
